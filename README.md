# Quiz Application

## Description
A simple interactive Quiz Application built with **HTML**, **CSS**, and **JavaScript**.
The app loads questions from a JSON file, displays them one by one, and lets the user select answers with a per-question countdown.

## Features
- **Countdown timer** for each question
- **Progress bullets** to track current question
- **Auto scoring** with final results (Good/Perfect/Bad)
- **Clean UI** with dynamic DOM rendering

## How to run
1. Open `index.html` directly in your browser.
2. Make sure `html_questions.json` is in the same directory (or update the path in `app.js`).
3. Start the quiz and view your results at the end.

## Technologies used
- HTML
- CSS
- JavaScript (DOM manipulation, events, JSON)
